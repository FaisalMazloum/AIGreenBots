# AIGreenBots: MSCA Project Workspace

[![EU Funding](https://img.shields.io)](https://aigreenbots.eu)

This is the central orchestration repository for the **AIGreenBots** project. It utilizes Git Submodules to maintain a hierarchical structure of Work Packages (WPs) and specific Deliverables.

---

## 📁 Project Structure (Work Package 3)

This workspace integrates the following core modules for USV swarm operations:

*   **[ROS2swarm_adjusted](./ROS2swarm_adjusted)**  
    *A specialized maritime fork of the `ros2swarm` framework, optimized for USV dynamics and AIGreenBots swarm behaviors.*
*   **[WP3-USV-Digital-Twin](./WP3-USV-Digital-Twin)**  
    *The Digital Twin synchronization layer, bridging physical vessel telemetry with Gazebo/Ignition simulations.*
*   **[WP3-Self-Diagnosis-Toolkit](./WP3-Self-Diagnosis-Toolkit)**  
    *Autonomous health monitoring and fault detection algorithms for resilient USV maritime swarms.*
